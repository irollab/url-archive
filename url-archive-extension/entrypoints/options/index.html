AI 智能处理（BYOK）

自带 API Key，剪藏时自动生成摘要、标签、关键词与回访场景。兼容 OpenAI 格式端点。

Obsidian 写入

选择剪藏落盘的通道。推荐使用 URL Archive 官方插件，无需额外安装 Local REST API。

在 Obsidian 的 URL Archive 插件设置中开启「剪藏服务」，端口需一致。

通用

剪藏笔记在 vault 中的存放目录。
